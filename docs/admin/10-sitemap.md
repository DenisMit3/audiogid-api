# Sitemap & Navigation

## Admin Panel (`/admin`)

- **Dashboard** (`/dashboard` or `/analytics/overview`)
    - Overview metrics (DAU, Revenue)
    - Recent Trends
- **Content**
    - **POIs** (`/admin/pois`): List, Search, Create, Edit.
    - **Tours** (`/admin/tours`): List, Builder, Publish.
- **Analytics**
    - **Overview** (`/analytics/overview`)
    - **Cohorts** (`/analytics/cohorts`): Retention Matrix.
    - **Funnels** (`/analytics/funnels`): Conversion Charts.
- **System**
    - **Users** (`/users`): Manage roles and status.
    - **Audit** (`/audit`): View logs and changes.
