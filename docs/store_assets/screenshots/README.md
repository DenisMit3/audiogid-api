# Store Screenshots

Please add screenshots here before submission.

## Requirements

### iOS (App Store) (RGB color space, no transparency, JPG/PNG)
* 6.5" iPhone (1284 x 2778 pixels) - Up to 10
* 5.5" iPhone (1242 x 2208 pixels) - Up to 10
* 12.9" iPad Pro (2048 x 2732 pixels) - Up to 10

### Android (Google Play) (JPEG or 24-bit PNG, no alpha)
* Phone: 1080 x 1920 pixels (portrait) or 1920 x 1080 (landscape)
* 7-inch tablet: 1600 x 2560 pixels (optional)
* 10-inch tablet: 1600 x 2560 pixels  (optional)

## Recommended Screenshots
1. **Explore**: Show the map with pins.
2. **Catalog**: Show the list of tours.
3. **Detail**: Show a POI detail screen with "Listen" button.
4. **Player**: Show the audio player screen.
5. **Offline**: Show the download manager.
