## Description (4000 chars max)
Откройте для себя Калининград с уникальным аудиогидом!

Audiogid — это ваш персональный экскурсовод в кармане. Исследуйте город в своем темпе, слушайте увлекательные истории о достопримечательностях и находите скрытые жемчужины.

Основные возможности:
- **Уникальные аудиотуры**: Профессионально озвученные экскурсии по историческим местам.
- **Оффлайн режим**: Скачивайте туры заранее и экономьте трафик в путешествии.
- **GPS навигация**: Удобная карта с маршрутами и автоматическим запуском историй.
- **QR-коды**: Сканируйте коды в музеях и на объектах для быстрого доступа к информации.
- **Каталог мест**: Сотни достопримечательностей с фотографиями и описанием.

Путешествуйте умно. Путешествуйте с Audiogid.
